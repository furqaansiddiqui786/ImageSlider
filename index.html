This is the Twenty-Twenty Effect below

How does this works?

To use this effect in your website, you must download the twenty-twenty library for javascript from the given download link below and follow the steps

  • First download the twentytwenty js library from here Download
  • Now Extract all the files
  • From the extracted file copy these required files into your website
    1. from the css folder copy twentytwenty.css
    2. from js folder copy jquery.twentytwenty and jquery.event.move
  • Now that we have all the files in our website
  • link our css in the head tag and scripts in body tags at the end

Now that we have all the required files and linked up to our html file

now we can design our image slider and add some styles and properties

  • Create a div.container
  • In the container create a div.jumbotron
  • In our jumbotron create a row and a single column of col-xs-12
  • We apply twentytwenty-container
  • In this column we add two images for before and after effect

In CUstom Style part we can give the col-xs-12 width of 100%

In our Custom written js file we can write

$(window).load(function() {
$(".col-xs-12").twentytwenty();
});
In the above function we target the column and apply the twenty-twenty effect to it

Thus the above steps shows how we can apply this effect in our websites
